# TouchDesigner

Courte description

Environnement node-based (Derivative TouchDesigner) pour la création visuelle temps réel, interactif et audiovisuel, souvent utilisé pour installations, projections et spectacles.

Usages typiques

- Mapping vidéo, shows en live, installations interactives.
- Prototypage d'interfaces visuelles complexes pilotées par capteurs.

Avantages

- Très puissant pour visuels temps réel et intégration de capteurs.
- Interface visuelle et workflow orienté live-coding.

Inconvénients

- Version non-open-source pour usage commercial (licences payantes pour certaines fonctionnalités).
- Courbe d'apprentissage pour notions 3D avancées.

Portabilité / Performance

- Fonctionne bien sur desktop; performances liées au GPU. Export/déploiement dépend de la machine hôte.

Quand choisir

- Installations visuelles haute-fidélité, événements live ou quand on a besoin d'une intégration visuelle et capteurs rapide.
